# LYN

one sauna. seven modules. twentyonehundred sites.

## Structure

```
/
├── index.html        # Home - centered LYN. logo
├── sauna.html        # Sauna page with module explorer modal
├── spa.html          # Spa page
├── residence.html    # Residence page  
├── onepager.html     # About/thesis
├── collaborate.html  # Partnership info
├── documents.html    # Document index
└── images/           # Image assets
    ├── config-preview.png
    ├── m1-sauna.png
    ├── m2-bridge.png
    ├── m3-changing.png
    ├── m4-thermal.png
    ├── m5-shower.png
    ├── m6-storage.png
    ├── m7-plunge.png
    ├── spa.png
    └── residence.png
```

## Deploy

Hosted on Netlify at lyn.page

## Images

Add images to the `/images` folder. Required:
- `config-preview.png` - sauna page main image
- `m1-sauna.png` through `m7-plunge.png` - module images for modal
- `spa.png` - spa page image
- `residence.png` - residence page image
